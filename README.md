# ASL Real-Time Recognition with Text-to-Speech
**Overview** </br>
This project implements a real-time American Sign Language (ASL) recognition system using a trained neural network model and MediaPipe for hand detection.</br>
**Features**</br>
Real-time ASL recognition using a webcam.
Predicts ASL letters and some words (e.g., 'del', 'nothing', 'space').
Displays the predicted sign on the video feed.
Speaks the predicted sign using text-to-speech.</br>
**Requirements**</br>
Python 3.x</br>
OpenCV</br>
TensorFlow</br>
MediaPipe</br>
NumPy</br>
pyttsx3</br>
**Installation**</br>
Clone the repository</br>
Install the required Python packages (requirements.txt)</br>
